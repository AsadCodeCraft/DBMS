## Backup & Restore

- **Use this command to Create Databse Backup**
command: 
```sql
  mysqldump -u root -p database_name > backup_file_name.sql
```
- **Use this command to Restore Databse**
command: 
```sql
  mysql -u root -p database_name < backup_file_name.sql
```
